# Last 30 Days Research Skill

## Overview

This skill instructions the agent to monitor recent trends and discussions over the past 30 days. It uses insights to inform decisions, such as creating mockups based on current prompting patterns or gathering the latest AI news.

## Resources

- **Skill Source**: [last30days-skill](https://github.com/mvanhorn/last30days-skill)
- **Framework**: [Eggent](https://github.com/eggent-ai/eggent)

## How to use

1. Search recent discussions, articles, and trends up to the last 30 days.
2. Use this data to validate the relevance of current tools or design patterns before acting on a user request.
3. Keep the user informed about newly identified patterns or tools that solve their problem more efficiently.
