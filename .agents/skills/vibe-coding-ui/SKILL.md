# Vibe Coding UI Skill

## Overview

This skill focuses on creating stunning, premium UI designs (Brutalism, Apple-style, etc.) through Vibe Coding. It relies on advanced design generation, utilizing visual references and tools like Google Flow, Whisk, and Antigravity.

## Resources

- **Design Inspiration**: [Bentogrids](https://bentogrids.com/), [Mobbin](https://mobbin.com/), Dribbble
- **Vibe Coding Tools**: [Supahero](https://www.supahero.io/), [Google Flow](https://labs.google/flow/about), [Google Whisk](https://labs.google/fx/tools/whisk)
- **MCP Servers**: [Playwright MCP](https://github.com/microsoft/playwright-mcp), [Figma MCP](https://www.framelink.ai/)

## How to use

1. Use visual references (from Dribbble or similar) or target websites using Playwright MCP to capture the desired "vibe".
2. Convert the design into production-ready UI components using Figma MCP merged with Shadcn components.
3. Automatically iterate on the design variants within Cursor to achieve visual perfection and smooth animations.
